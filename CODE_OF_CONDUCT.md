# 社区行为准则

## 我们的承诺

作为贡献者和维护者，我们承诺让 pm-agent-toolkit 的参与过程对所有人友善、尊重、专业 —— 无论年龄、身体状况、可见或不可见的残疾、民族、性别特征、性别认同和表达、经验水平、教育、社会经济地位、国籍、个人外貌、种族、宗教或性取向。

## 行为准则

### 鼓励

- **就事论事的讨论**：对事不对人，对方案不对身份
- **先读文档再提问**：90% 的问题答案在 `docs/` 和 README 里
- **坦诚表达不同意见**：不认同某个设计？说出为什么，附上替代方案
- **尊重 opinionated 选择**：本项目的范式是 opinionated 的，如果你的团队风格不同，鼓励 fork 而不是拉扯
- **帮助新人**：Issue 提得不标准？友好引导对方改好

### 不被接受

- 人身攻击、侮辱性言论、歧视
- 性暗示或任何形式的骚扰
- 发布他人的私人信息
- 在公开场合质疑维护者的动机
- "我的公司就是这么做的，你们错了" 式的傲慢

## 执行

维护者有权：

- 编辑或删除不符合本准则的评论、commit、代码、wiki、issue 等
- 临时或永久封禁任何违反准则的贡献者

违反本准则的行为可以私信维护者 xiuqi@logic-labs.ai 报告。

## 归属

本准则参考 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html)。
